Steps to run the .jar file: 
1. open command prompt 
2. navigate to the jar file (for example: cd desktop/swrve_test/jar)
3. run jar file and specify the url as argument (java -jar swrve.jar urlwheregzfileislocated)

The project was built in the Eclipse Ide on the Java platform, to open select "open projects from file system".
There are 10 test cases implemented using the JUnit framework.
1 external library is used for handling dates and times(Joda-Time).
